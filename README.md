CRUD de Clientes – Projeto de Estudo em C

Este é um projeto de estudo em C, desenvolvido para praticar conceitos de estrutura de dados utilizando listas estáticas (arrays). O sistema funciona como um CRUD, permitindo cadastrar, atualizar, buscar e remover clientes, aplicando os fundamentos de manipulação de registros e organização de código.

🛠 Tecnologias

Linguagem C

Manipulação de arrays e structs

Funções para operações CRUD

📌 Funcionalidades

Criar clientes com código, nome, CPF, telefone e endereço

Ler/buscar clientes pelo código

Atualizar informações de clientes

Deletar clientes pelo código

Listar todos os clientes cadastrados

🔍 Objetivo de aprendizado

Entender como dados são armazenados na memória

Praticar manipulação de arrays e structs

Construir um CRUD funcional do zero
